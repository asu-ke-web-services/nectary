Nectary\Factories\View_Factory::add_data

Chainable

Inject data into the view. You can pass an array, or named array. This data will be added to the view's data model. All of the following are valid data: Inject the $event into the 'event' context: ``` $view->add_data( array( 'event' => $event->present() ) ); ``` Inject the $event into the global context: ``` $view->add_data( array( '_event' => $event->present() ) ); ``` Inject the $event into the global context: ``` $view->add_data( $event->present() ); ```
